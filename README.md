# 💼 JOB APPLICATION TRACKER

This is a simple web application built to help track job applications in an organized way.  
It allows users to add, view, update, and remove job entries directly from the browser using local storage.

The goal of this project was to practice frontend development using HTML, CSS, and JavaScript while building something practical.
## 🚀 Features

- Add job applications with company name, role, and status  
- View all applications in a structured layout  
- Update or modify existing entries  
- Delete applications when no longer needed  
- Data is stored locally in the browser (no database required).
